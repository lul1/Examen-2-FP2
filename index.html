1. ¿Que es POO?

POO o Programacion Orientada a Objetos es un estilo de programacion, que se acerca a la manera de como nostros expresamos las cosas en vida real. Es por esto que este metodo es mas facil de entender ya que simplifica el problema dejandonos ver solo lo importante, ademas que que su modificabilidad es muy util a la hora de hacer cambios en el problema

2. Explica la diferencia entre Clase y Objeto

La diferencia es que sin una clase no puede haber un objeto, siempre, primero, debe crearse la clase, entonces una ves creada la clase un objeto puede pertencer a dicha clase. por ejemplo el curso de programacion 2 es un objeto dentro de la clase que puede ser todos los cursos que pueden existir.

3. ¿Que Git y Github.? Explique la diferencia

Git es un software de control de versionamiento, que sirve para la administracion de una aplicacion que tiene bastantes codigos. Github es una plataforma que almacena proyectos a traves de repositorios. La diferencia es que uno no requiere de otro, github es una plataforma que te brinda el servicio de hacer proyectos usando el git y el poder publicarlos pero tambien puede ser usado sin este a la ves que git, puede ser usado sin github.
